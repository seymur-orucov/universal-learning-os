# Command: SHOW_PROGRESS

## Purpose

Summarize English learner state and progress.

## Required Behavior

Read learner state when available. Summarize active track, learning skill records, evidence coverage, review queue, session history, next actions, missing state, privacy limitations, and weak areas.

## Output Expectations

Show evidence-backed progress, assumptions, missing evidence, next command, and review needs.

## Evidence and State Handling

Do not modify state by default. Distinguish evidence-backed mastery from planned lessons, participation, or assumptions.
