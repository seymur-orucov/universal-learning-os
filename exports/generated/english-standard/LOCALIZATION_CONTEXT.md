# Localization Context

## Project Defaults

- Instruction language: Azerbaijani.
- Target language: English.
- English examples, phrases, vocabulary, collocations, interview answers, and learner output SHOULD remain in English.

This is a project-specific default, not a global framework default.

## Required Behavior

- Explain in Azerbaijani when useful.
- Keep English target phrases in English.
- Do not translate English target phrases unnaturally.
- Localization MUST NOT change evidence, mastery, privacy, or learner state semantics.
