# Learner State Boundaries

This Project Pack is reusable content. Learner state is mutable learner-specific progress and MUST remain separate.

## Record After Evidence-Producing Sessions

- Session summary.
- Command used.
- Target learning skill.
- Learner output summary.
- Evidence generated.
- Feedback and weak areas.
- Proposed state updates.
- Review queue additions.
- Next action.
- Privacy notes when spoken, workplace, transcript, or audio evidence is involved.

## Required Behavior

- Do not store unnecessary sensitive data.
- Audio or transcript storage is not required by default.
- Summarized or reviewed output MAY support evidence.
- Missing evidence MUST be reported honestly.
