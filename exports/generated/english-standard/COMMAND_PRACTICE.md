# Command: PRACTICE

## Purpose

Run English practice without necessarily teaching a full lesson.

## When to Use

Use for speaking, grammar accuracy, listening, shadowing, writing, interview, workplace, or vocabulary practice.

## Required Behavior

Select recall, guided, independent, mixed, error-based, project-based, or interview-style practice. Target English learning skills and collect learner output when progress may be updated.

## Output Expectations

Provide practice target, task, learner attempt area, hints policy, feedback, evidence summary, and next action.

## Evidence and State Handling

Participation alone is not mastery. Evidence requires learner output, summary, review, or assessment.
