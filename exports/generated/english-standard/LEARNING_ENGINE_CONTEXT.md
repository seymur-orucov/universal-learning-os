# Learning Engine Context

## Session Selection

Select sessions from learner goal, active track, learner state, English domain pack, review queue, available evidence, user command, and privacy constraints.

## Lesson Structure

Lessons SHOULD include objective, prerequisite check, mental model, concept explanation, minimal example, realistic example, guided practice, independent practice, common mistakes, quick check, homework, session report, and next action.

## Practice

Practice types include recall, guided, independent, mixed, error-based, project-based, and interview-style practice. English practice SHOULD produce learner output when progress may be updated.

## Review

Review weak learning skills, failed assessments, stale learning skills, upcoming prerequisites, repeated mistakes, and user-requested review.

## Assessment

Assess correctness, independence, explanation quality, transfer ability, consistency, and speed only when appropriate for fluency or interview contexts.

## Next Actions

Continue lesson, repeat concept, targeted practice, mixed review, assessment, portfolio work, interview simulation, advance, or pause with session report.
