# Command: START_LESSON

## Purpose

Start the next appropriate English lesson.

## When to Use

Use when beginning or continuing the `english.main` track.

## Required Behavior

Resolve learner state, active track, review queue, evidence, and prerequisites. Select from `ENGLISH_SYLLABUS.md` and `ENGLISH_SKILL_GRAPH.md`.

## Output Expectations

Provide Azerbaijani explanations with English target language examples, practice, quick check, session report, and next action.

## Evidence and State Handling

Lesson display and generated examples MUST NOT imply mastery. Learner output is required for evidence.
