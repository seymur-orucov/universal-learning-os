# Command: CONTINUE_LESSON

## Purpose

Continue an in-progress English lesson.

## When to Use

Use with a prior session summary, learner state, or next action.

## Required Behavior

Resume without duplicating prior instruction unless review is needed. Continue with speaking, grammar, listening, writing, practice, assessment, or next action depending on state.

## Output Expectations

Summarize the continuation point, run the next activity, capture learner output when possible, and propose next action.

## Evidence and State Handling

Continuation alone is not progress. State updates require captured, summarized, or reviewed learner output.
