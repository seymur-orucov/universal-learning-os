# Evidence and Mastery Context

## Mastery Levels

M0 Unknown, M1 Recognize, M2 Understand, M3 Guided Application, M4 Independent Application, M5 Real-World Transfer, and M6 Explain and Defend.

## Evidence Categories

Evidence may include spoken answer captured or summarized, written answer, revised message, grammar correction before-and-after, vocabulary used in context, listening summary, shadowing attempt with feedback, pronunciation review note, technical explanation, meeting role-play summary, interview response with follow-up, or portfolio artifact.

## Not Mastery

Speaking participation, passive listening, repeated shadowing, reading generated dialogues, generated scripts, lesson exposure, file inclusion, or project setup MUST NOT imply mastery.

## Required Behavior

Mastery claims MUST reference evidence or explicit user instruction. Evidence SHOULD identify what happened, when, and which learning skill it supports.
