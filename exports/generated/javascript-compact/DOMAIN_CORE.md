# JavaScript Domain Core

Canonical sources: `domains/javascript/DOMAIN.md`, `SYLLABUS.md`, `SKILL_GRAPH.md`, `PRACTICE_RULES.md`, `ASSESSMENT_RULES.md`, `PROJECTS.md`, `GLOSSARY.md`, and supporting JavaScript domain files.

## Goal

Help the learner write, debug, test, refactor, optimize, and explain practical JavaScript behavior for frontend/backend engineering and interviews.

## Track

- Beginner foundations: `var`, `let`, `const`, primitives/references, operators, conditionals, loops, functions, arrays, objects, DOM basics, debugging.
- Mental models: execution context, call stack, scope chain, lexical environment, hoisting, closures, `this`, prototypes, classes, property descriptors, mutability.
- Async JavaScript: callbacks, promises, `async/await`, microtasks, macrotasks, event loop, timers, `fetch`, `AbortController`, async errors.
- Modern JavaScript: modules, destructuring, spread/rest, optional chaining, nullish coalescing, iterators, generators, `Map`, `Set`, `WeakMap`, `WeakSet`, symbols.
- Professional readiness: memory, garbage collection basics, event delegation, debounce/throttle, cloning, functional patterns, composition, module design, performance, rendering, Web APIs, XSS/CSRF awareness, testing, interviews.

## Evidence

Valid evidence includes reviewed learner code, predictions, explanations, bug fixes, refactors, tests, async traces, performance reasoning, safe DOM/security reasoning, and frontend/backend scenario solutions. Generated examples are not evidence by themselves.
