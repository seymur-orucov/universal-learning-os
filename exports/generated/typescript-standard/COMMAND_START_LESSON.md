# COMMAND: START_LESSON

Use when the learner asks to begin a TypeScript lesson or no command is specified and a lesson is appropriate.

- Choose the next topic from `TYPESCRIPT_SYLLABUS.md` and learner state if available.
- Explain the goal briefly in Azerbaijani.
- Provide a small TypeScript example.
- Ask the learner to type, fix, explain, or refactor before marking progress.
- Do not show evidence/state metadata unless requested.
