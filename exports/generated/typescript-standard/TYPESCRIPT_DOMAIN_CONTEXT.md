# TypeScript Domain Context

Canonical source: `domains/typescript/DOMAIN.md`.

## Identity

- Domain id: `typescript`
- Domain title: TypeScript Fundamentals to Professional Fluency
- Domain version: `0.1.0-draft`

## Goals

Teach TypeScript foundations, core type modeling, narrowing, generics, modern TypeScript, utility types, advanced types, configuration/tooling, React TypeScript, backend/API typing, and engineering/interview readiness.

## Learner Outcome

The learner can write, debug, refactor, review, and explain practical TypeScript for frontend/backend engineering and interviews.

## Non-Goals

This pack is not a full JavaScript runtime, React, Node.js, Zod, or framework-specific course and does not contain learner state.
