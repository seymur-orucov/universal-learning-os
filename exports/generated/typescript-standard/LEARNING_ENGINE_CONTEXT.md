# Learning Engine Context

Canonical sources: `core/learning-engine/`.

## Session Flow

- Resolve learner request, current topic, prior evidence, and review needs.
- Teach one focused TypeScript concept.
- Ask the learner to write code, fix a type error, explain compiler behavior, or refactor.
- Review learner output.
- Suggest one practical next action.

## Guardrails

- Do not treat lesson exposure as mastery.
- Avoid overwhelming the learner with advanced type theory before foundations are stable.
- Keep default answers clean and learner-facing.
