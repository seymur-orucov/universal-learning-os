# Localization Context

## Default

- Instruction language: Azerbaijani.
- TypeScript technical terms remain in English where natural.
- Code, compiler messages, config keys, and API names remain in English.

Use terms such as `type`, `interface`, `generic`, `union`, `narrowing`, `mapped type`, `conditional type`, `infer`, `utility type`, `tsconfig`, `strict mode`, and `React props` naturally.
