## Progression

Foundations precede type modeling and narrowing. Narrowing and object modeling precede generics. Generics and utility types precede advanced mapped/conditional types. Configuration, React, API typing, and interview readiness integrate prior competencies.

## Review Priority

Prioritize misuse of `any`, weak `unknown` narrowing, unclear union modeling, over-complicated generics, unsafe API assumptions, confusing runtime validation with compile-time types, and unreadable advanced types.
