# TypeScript Glossary and Projects

Canonical sources: `domains/typescript/GLOSSARY.md` and `domains/typescript/PROJECTS.md`.

## Glossary Guidance

- `type`: type alias yaratmaq üçün keyword.
- `interface`: object shape təsvir etmək üçün construct.
- `generic`: reusable type/function/component üçün type parameter.
- `union`: bir neçə mümkün type-dan biri.
- `narrowing`: union və ya `unknown` type-ı runtime check-lə konkretləşdirmək.
- `mapped type`: key-lər üzərində type transformation.
- `conditional type`: type-level branching.
- `infer`: conditional type daxilində type çıxarmaq.
- `utility type`: `Partial`, `Pick`, `Omit`, `ReturnType`, və s. kimi built-in helper.
- `tsconfig`: TypeScript compiler configuration faylı.
- `strict mode`: daha sərt type checking flag-ləri.
- `React props`: React component input-larının type-lanması.

## Project Ideas

- Convert JavaScript utilities to TypeScript.
- Model API DTOs and response/error results.
- Build typed React component props and form handling.
- Refactor `any` to `unknown`, generics, or explicit domain types.
- Create generic helpers such as `getByKey`, `groupBy`, and typed API wrappers.
- Review bad TypeScript and explain safety/readability tradeoffs.

Project work is evidence only when learner code or explanation is reviewed.
