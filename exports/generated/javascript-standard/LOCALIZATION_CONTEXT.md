# Localization Context

## Default

- Instruction language: Azerbaijani.
- JavaScript technical terms remain in English where natural.
- Code remains in JavaScript.

## Style

Use concise Azerbaijani explanations with English technical terms such as `scope`, `closure`, `this`, `prototype`, `Promise`, `async/await`, `event loop`, `module`, and `DOM`.

Do not over-translate API names, syntax, runtime terms, or browser concepts.
