# JavaScript Domain Context

## Identity

- Domain id: `javascript`
- Domain title: JavaScript Fundamentals for Frontend Engineers
- Domain version: `0.1.0-draft`

## Goals

Teach JavaScript from zero/fundamentals through professional fluency: variables, primitives/references, operators, control flow, functions, arrays, objects, DOM basics, debugging, execution context, call stack, scope chain, lexical environment, hoisting, closures, `this`, prototypes, classes, property descriptors, mutability, async behavior, promises, event loop, modules, modern language features, browser/Web APIs, performance, memory, security basics, testing, and interview reasoning.

## Learner Outcome

The learner can write, debug, test, refactor, optimize, and explain practical JavaScript behavior for frontend/backend engineering and interviews.

## Non-Goals

This pack is not a full React, TypeScript, framework-specific, or platform-deep browser/Node.js API course and does not contain learner state.
