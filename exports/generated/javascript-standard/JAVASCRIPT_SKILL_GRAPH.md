## Progression

Values and control flow precede functions. Functions precede execution context, scope chain, lexical environment, hoisting, and closures. Objects and arrays precede property descriptors, prototypes, classes, and immutability patterns. Scope, functions, and objects support `this` reasoning. Promises and async/await precede event loop, fetch, cancellation, and async error reasoning. Browser/Web APIs, rendering, performance, memory, security, testing, and interview reasoning integrate prior competencies.

## Review Priority

Prioritize review for repeated mistakes in equality/coercion, closure, `this`, mutation/copying, prototype lookup, promise chains, async error handling, event loop ordering, memory leaks, unsafe DOM handling, performance reasoning, and unclear explanations.
