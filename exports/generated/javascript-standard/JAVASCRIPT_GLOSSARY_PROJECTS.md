# JavaScript Glossary and Projects

## Glossary Guidance

- `scope`: dəyişənin haradan əlçatan olduğunu göstərən qayda.
- `closure`: function-un öz lexical scope məlumatını saxlaması.
- `this`: çağırılma formasından asılı olan context dəyəri.
- `prototype`: obyektlər arasında method/property lookup mexanizmi.
- `Promise`: async nəticəni təmsil edən obyekt.
- `event loop`: call stack, task queue və microtask queue arasında icra koordinasiyası.
- `module`: import/export ilə ayrılmış kod vahidi.
- `DOM`: browser-in HTML sənədini obyekt modeli kimi təqdim etməsi.
- `AbortController`: async request və ya operation-u cancellation etmək üçün Web API.
- `debounce`: ardıcıl çağırışları gözlədib yalnız son çağırışı icra etmək pattern-i.
- `throttle`: müəyyən intervalda maksimum bir dəfə icra etmə pattern-i.
- `reflow/repaint`: browser rendering zamanı layout və paint yenilənmələri.
- `XSS`: unsafe DOM/input handling nəticəsində script injection riski.
