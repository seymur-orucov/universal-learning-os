# COMMAND: START_LESSON

Use when the learner asks to begin a JavaScript lesson or no command is specified and a lesson is appropriate.

## Behavior

- Choose the next topic from `JAVASCRIPT_SYLLABUS.md` and learner state if available.
- Explain the goal briefly in Azerbaijani.
- Provide a small JavaScript example.
- Ask the learner to predict, explain, or modify before marking progress.
- Do not show evidence/state metadata unless requested.
