# JavaScript Syllabus

## Track: `javascript.main`

1. Beginner foundations: `var`, `let`, `const`, primitive vs reference types, operators, expressions, conditionals, loops, functions, arrays, objects, basic DOM usage, DevTools, basic debugging.
2. Values, equality, and data behavior: identity, `==` vs `===`, coercion, truthy/falsy values, assignment, references, shallow copy, shared mutation, basic immutability.
3. Functions and abstractions: declarations, expressions, arrow functions, parameters, returns, callbacks, pure functions, side effects, higher-order functions.
4. Execution context and scope: execution context, call stack, scope chain, lexical environment, hoisting, temporal dead zone, block/function/global scope, shadowing.
5. Closures and practical state: lexical capture, private state, factories, loop closure traps, closures and memory leaks, stale closure scenarios.
6. Objects, arrays, and property behavior: destructuring, spread/rest, optional chaining, nullish coalescing, template literals, modern methods, property descriptors, mutability, deep copy vs shallow copy, structured cloning.
7. `this`, prototypes, and classes: binding rules, method calls, detached methods, `call`, `apply`, `bind`, constructors, prototype chain, prototypal inheritance, classes as syntax over prototypes, instance/static methods.
8. Modern JavaScript: ES modules, module boundaries, iterators, generators, `Map`, `Set`, `WeakMap`, `WeakSet`, symbols, collection tradeoffs.
9. Async JavaScript: callbacks, promises, promise chaining, `async/await`, microtasks vs macrotasks, event loop, timers, `fetch`, `AbortController`, async error handling, race conditions.
10. Browser runtime and Web APIs: DOM updates, events, event delegation, forms, browser rendering basics, reflow/repaint, storage APIs, safe DOM handling.
11. Professional engineering: module design, composition, functional programming patterns, immutability patterns, application-level error boundaries, performance optimization, debouncing, throttling, memory management, garbage collection basics, testing JavaScript logic.
12. Security and robustness: XSS awareness, CSRF awareness, validation, escaping/sanitization concepts, dependency awareness, error reporting boundaries.
13. Engineering and interview readiness: common JavaScript interview questions, explain-output tasks, bug fixing, refactoring, real frontend scenarios, practical code reading, mental model questions, performance reasoning, async behavior tracing.

Each lesson should include explanation, one small example, learner prediction or code, review, and next practice suggestion.
