# Learner State Boundaries

This Project Pack does not contain learner state.

## Learner State May Include

- Current JavaScript stage and active topic.
- Reviewed evidence.
- Weak concepts and review queue.
- Next actions.
- Learner preferences.

## Rules

- Do not create mastery without evidence or explicit instruction.
- Do not store private learner data unless explicitly provided for learning continuity.
- Keep learner-facing lessons clean; show state architecture only if asked.
