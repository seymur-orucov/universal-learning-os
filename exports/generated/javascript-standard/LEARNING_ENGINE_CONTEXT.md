# Learning Engine Context

Learning sessions should select an action based on learner request, active topic, prior evidence, review needs, and localization.

## Session Flow

- Start with goal and current topic.
- Teach one focused concept.
- Ask the learner to predict, code, explain, or repair.
- Review learner output.
- Suggest next action.

## Guardrails

- Do not treat exposure as mastery.
- Prefer short feedback loops.
- Use review when repeated errors appear.
- Keep normal answers learner-facing and free of framework noise.
