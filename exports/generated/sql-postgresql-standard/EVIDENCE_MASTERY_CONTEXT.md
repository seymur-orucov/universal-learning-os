# Evidence and Mastery Context

## Mastery Scale

- M0 Unknown: no reliable evidence.
- M1 Recognize: identifies or distinguishes the concept.
- M2 Understand: explains concept or reasoning.
- M3 Guided Application: applies with support.
- M4 Independent Application: applies without material support in familiar contexts.
- M5 Real-World Transfer: applies in realistic, novel, or integrated contexts.
- M6 Explain and Defend: justifies decisions and responds to challenge.

## Evidence Categories

Mastery claims SHOULD be based on learner output, reviewed answer, assessment result, project artifact, interview response, or explicit user instruction.

## What Does Not Count as Mastery

- Reading a lesson.
- Seeing generated examples.
- Uploading files into a project.
- Completing setup.
- Receiving an explanation.
- Producing an artifact that has not been reviewed.

## Required Behavior

- Mastery claims MUST reference evidence or explicit user instruction.
- Evidence SHOULD identify what the learner did, when it happened, and which learning skill it supports.
- Domain rubrics MUST NOT weaken evidence requirements.
- State updates SHOULD distinguish demonstrated evidence from explicit user instruction.
