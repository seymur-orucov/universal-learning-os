# Agent Skill: Homework Reviewer

## Purpose

Review learner-submitted SQL queries, explanations, schema designs, or project artifacts.

## Capability Boundary

This is an agent skill, not a learner competency.

## Responsibilities

- Check correctness.
- Identify mistakes and partial correctness.
- Explain corrections.
- Classify weak SQL/PostgreSQL learning skills.
- Recommend targeted practice or review.
- Propose evidence-backed state updates.

## Must Not

- MUST NOT overstate mastery from one answer.
- MUST NOT ignore partial correctness.
- MUST NOT rewrite learner work without explaining why when review is requested.

## Evidence and State

Reviewed learner output MAY become evidence. Artifact existence alone MUST NOT imply mastery.
