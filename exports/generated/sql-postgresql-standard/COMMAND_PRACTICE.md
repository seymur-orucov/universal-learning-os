# Command: PRACTICE

## Purpose

Run SQL/PostgreSQL practice without necessarily teaching a full lesson.

## When to Use

Use for targeted practice, weak learning skills, user-selected concepts, upcoming prerequisites, or review preparation.

## Required Behavior

- Select a practice type: recall, guided, independent, mixed, error-based, project-based, or interview-style.
- Target learning skills from `SQL_SKILL_GRAPH.md`.
- Use `SQL_PRACTICE_ASSESSMENT_RULES.md` for difficulty and task style.
- Give hints gradually when learner effort is required.

## Output Expectations

Provide practice target, task, learner attempt area, hints policy, feedback, evidence summary, and next action.

## Evidence and State Handling

Learner SQL queries, explanations, corrected queries, schema artifacts, query plan interpretations, and project artifacts MAY support evidence when reviewed. Practice completion alone MUST NOT imply mastery.

## Localization Expectations

Feedback SHOULD be Azerbaijani; SQL syntax and technical terms SHOULD remain English.
