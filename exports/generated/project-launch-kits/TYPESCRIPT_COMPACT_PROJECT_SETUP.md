# TypeScript Compact Project Setup

## Project Name

TypeScript Mentor OS Compact

## Project Purpose

Teach TypeScript in a Free-plan ChatGPT Project using a max-5-file compact pack.

## Profile

Use the compact pack for Free Project usage or other 5-file-limit environments.

## Step 1: Create Project

Create a new ChatGPT Project named `TypeScript Mentor OS Compact`.

## Step 2: Add Project Instructions

Paste `exports/generated/typescript-compact/PROJECT_INSTRUCTIONS.md` into Project Instructions.

## Step 3: Upload Compact Files

Upload the remaining files from `exports/generated/typescript-compact/` according to `TYPESCRIPT_COMPACT_UPLOAD_FILE_LIST.md`.

## Step 4: Start First Chat

Use `exports/generated/project-launch-kits/TYPESCRIPT_COMPACT_STARTUP_PROMPT_READY.md`.

## Warnings

- Compact packs summarize aggressively to fit 5 files.
- Learner state remains outside the Project Pack.
- Normal learner-facing answers should not show evidence/state metadata unless requested.
