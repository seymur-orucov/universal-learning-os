# TypeScript Project Setup

## Project Name

TypeScript Mentor OS

## Project Purpose

Teach TypeScript from foundations to professional fluency for frontend/backend engineering, React, API typing, large codebases, and interviews.

## Profile

Use the standard pack for Plus/Go or higher Project usage when 25 files are available.

## Step 1: Create New ChatGPT Project

Create a new ChatGPT Project named `TypeScript Mentor OS`.

## Step 2: Add Project Instructions

Paste or adapt `exports/generated/typescript-standard/PROJECT_INSTRUCTIONS.md` into Project Instructions.

## Step 3: Upload Source Files

Upload source files from `exports/generated/typescript-standard/`. If Project Instructions are handled separately, upload the remaining 24 files as knowledge/source files.

## Step 4: Start First Chat

Start the first chat using `exports/generated/project-launch-kits/TYPESCRIPT_STARTUP_PROMPT_READY.md`.

## Step 5: Continue Future Chats

Continue future chats using `exports/generated/project-launch-kits/TYPESCRIPT_CONTINUATION_PROMPT_READY.md` with the previous session summary.

## Step 6: Maintain Learner State Separately

Keep learner progress, evidence, weak TypeScript topics, review queue, privacy notes, and next actions outside the Project Pack.

## Localization Notes

Use Azerbaijani explanations while preserving TypeScript technical terms in English where natural.
