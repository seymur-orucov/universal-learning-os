# English Compact Project Setup

## Project Name

English Mentor OS Compact

## Project Purpose

Teach English for Software Engineers in a Free-plan ChatGPT Project using a max-5-file compact pack.

## Step 1: Create Project

Create a new ChatGPT Project named `English Mentor OS Compact`.

## Step 2: Add Project Instructions

Paste `exports/generated/english-compact/PROJECT_INSTRUCTIONS.md` into Project Instructions.

## Step 3: Upload Compact Files

Upload the remaining files from `exports/generated/english-compact/` according to `ENGLISH_COMPACT_UPLOAD_FILE_LIST.md`.

## Step 4: Start First Chat

Use the prompt from `exports/generated/english-compact/STARTUP_PROMPT.md`.

## Warnings

- Compact packs summarize aggressively to fit 5 files.
- Learner state remains outside the Project Pack.
- Normal learner-facing answers should not show evidence/state metadata unless requested.
