# Go Compact Project Setup

## Project Name

Go Mentor

Create one dedicated ChatGPT Project for Go and use exactly the five files from `exports/generated/go-compact/`. This compact profile is intended for the five-file ChatGPT Project limit. Paste `PROJECT_INSTRUCTIONS.md` into Project Instructions, upload the remaining files, and use `GO_COMPACT_STARTUP_PROMPT_READY.md` or simply type `Start lesson`.

Never upload compact and standard together. Generated files can be used directly from `exports/generated/`; run the generator only to regenerate artifacts after canonical-source changes.

ChatGPT Projects remain the normal learning runtime. Studio is optional. Compact keeps the same language-first, standard-library-first, teaching, concurrency, evidence, and production boundaries within five files.
