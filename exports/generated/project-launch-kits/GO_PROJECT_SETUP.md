# Go Project Setup

## Project Name

Go Mentor

Create one dedicated ChatGPT Project for Go. Use all 25 files from `exports/generated/go-standard/` when that file budget fits. Paste `PROJECT_INSTRUCTIONS.md` into Project Instructions, upload the remaining files, and start with `GO_STARTUP_PROMPT_READY.md` or simply type `Start lesson`.

Never upload the standard and compact profiles together. The generated files can be taken directly from `exports/generated/`; running the CLI generator is required only when regenerating packs or changing canonical sources.

ChatGPT Projects remain the normal learning runtime. Universal Learning OS Studio is optional and CLI-backed. Learner state stays separate, and handoff/snapshot helpers are used only when explicitly requested.
