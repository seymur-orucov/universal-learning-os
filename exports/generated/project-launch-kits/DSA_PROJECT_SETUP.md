# DSA Project Setup

## Project Name

DSA Mentor

Create one dedicated ChatGPT Project. Use `exports/generated/dsa-standard/` when the 25-file profile fits. Paste `PROJECT_INSTRUCTIONS.md` into Project Instructions, upload the remaining pack files, and start with `DSA_STARTUP_PROMPT_READY.md`.

JavaScript is the default implementation language. TypeScript is available when requested or useful for clarity. Keep learner state separate and use handoff or snapshot helpers only when explicitly requested.

Studio is optional and is not required for learning.
