# SQL/PostgreSQL Compact Project Setup

## Project Name

SQL/PostgreSQL Mentor OS Compact

## Project Purpose

Teach SQL and PostgreSQL in a Free-plan ChatGPT Project using a 5-file compact pack.

## Step 1: Create Project

Create a new ChatGPT Project named `SQL/PostgreSQL Mentor OS Compact`.

## Step 2: Add Project Instructions

Paste `exports/generated/sql-postgresql-compact/PROJECT_INSTRUCTIONS.md` into Project Instructions.

## Step 3: Upload Compact Files

Upload the remaining files from `exports/generated/sql-postgresql-compact/` according to `SQL_POSTGRESQL_COMPACT_UPLOAD_FILE_LIST.md`.

## Step 4: Start First Chat

Use the prompt from `exports/generated/sql-postgresql-compact/STARTUP_PROMPT.md`.

## Warnings

- Compact packs summarize aggressively to fit 5 files.
- Learner state remains outside the Project Pack.
- Normal learner-facing answers should not show evidence/state metadata unless requested.
