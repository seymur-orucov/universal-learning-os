# SQL/PostgreSQL Project Setup

## Project Name

SQL/PostgreSQL Mentor OS

## Project Purpose

Teach SQL and PostgreSQL step by step from beginner to advanced, including practical database work, Northwind-style tasks, normalization, indexing, transactions, MVCC, `EXPLAIN ANALYZE`, and query optimization.

## Step 1: Create New ChatGPT Project

Create a new ChatGPT Project named `SQL/PostgreSQL Mentor OS`.

## Step 2: Add Project Instructions

Paste or adapt `exports/generated/sql-postgresql-standard/PROJECT_INSTRUCTIONS.md` into Project Instructions.

## Step 3: Upload Source Files

Upload source files from `exports/generated/sql-postgresql-standard/`. If Project Instructions are handled separately, upload the remaining 24 files as knowledge/source files.

## Step 4: Start First Chat

Start the first chat using `exports/generated/project-launch-kits/SQL_POSTGRESQL_STARTUP_PROMPT_READY.md`.

## Step 5: Continue Future Chats

Continue future chats using `exports/generated/project-launch-kits/SQL_POSTGRESQL_CONTINUATION_PROMPT_READY.md` with the previous session summary.

## Step 6: Maintain Learner State Separately

Keep learner progress, evidence, weak learning skills, review queue, and next actions outside the Project Pack.

## Step 7: Save Session Summaries

After lessons or practice, save session summaries with evidence generated, proposed state updates, and next action.

## Localization Notes

Use Azerbaijani explanations. Keep SQL/PostgreSQL technical terminology in English where natural.
