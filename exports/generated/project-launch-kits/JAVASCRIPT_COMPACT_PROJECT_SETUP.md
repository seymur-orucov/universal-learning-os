# JavaScript Compact Project Setup

## Project Name

JavaScript Mentor OS Compact

## Project Purpose

Teach JavaScript from beginner foundations to professional fluency in a Free-plan ChatGPT Project using a 5-file compact pack.

## Step 1: Create Project

Create a new ChatGPT Project named `JavaScript Mentor OS Compact`.

## Step 2: Add Project Instructions

Paste `exports/generated/javascript-compact/PROJECT_INSTRUCTIONS.md` into Project Instructions.

## Step 3: Upload Compact Files

Upload the remaining files from `exports/generated/javascript-compact/` according to `JAVASCRIPT_COMPACT_UPLOAD_FILE_LIST.md`.

## Step 4: Start First Chat

Use the prompt from `exports/generated/javascript-compact/STARTUP_PROMPT.md`.

## Warnings

- Compact packs summarize aggressively to fit 5 files.
- Learner state remains outside the Project Pack.
- Normal learner-facing answers should not show evidence/state metadata unless requested.
