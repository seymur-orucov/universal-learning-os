# English Project Setup

## Project Name

English Mentor OS

## Project Purpose

Teach English for Software Engineers from approximately B1 toward C1 with speaking-first professional communication.

## Learner Goal

B1 to C1 progression with speaking, listening, vocabulary, grammar for accuracy, professional communication, daily communication, interview English, and technical English for software development.

## Step 1: Create New ChatGPT Project

Create a new ChatGPT Project named `English Mentor OS`.

## Step 2: Add Project Instructions

Paste or adapt `exports/generated/english-standard/PROJECT_INSTRUCTIONS.md` into Project Instructions.

## Step 3: Upload Source Files

Upload source files from `exports/generated/english-standard/`. If Project Instructions are handled separately, upload the remaining 24 files as knowledge/source files.

## Step 4: Start First Chat

Start the first chat using `exports/generated/project-launch-kits/ENGLISH_STARTUP_PROMPT_READY.md`.

## Step 5: Continue Future Chats

Continue future chats using `exports/generated/project-launch-kits/ENGLISH_CONTINUATION_PROMPT_READY.md` with the previous session summary.

## Step 6: Maintain Learner State Separately

Keep learner progress, evidence, weak learning skills, review queue, privacy notes, and next actions outside the Project Pack.

## Step 7: Save Session Summaries

After lessons or practice, save session summaries with evidence generated, proposed state updates, and next action.

## Localization Notes

Use Azerbaijani explanations with English examples, phrases, vocabulary, interview answers, and learner output where appropriate.

## Warnings

- Repeated reading or passive exposure does not equal speaking mastery.
- Generated dialogues, shadowing prompts, or scripts do not imply mastery.
- Do not store audio/transcripts unless explicitly authorized.
