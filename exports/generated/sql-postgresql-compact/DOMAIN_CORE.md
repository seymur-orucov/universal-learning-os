# SQL/PostgreSQL Domain Core

Canonical sources: `domains/sql-postgresql/DOMAIN.md`, `SYLLABUS.md`, `SKILL_GRAPH.md`, `PRACTICE_RULES.md`, `ASSESSMENT_RULES.md`, `PROJECTS.md`, and `GLOSSARY.md`.

## Goal

Help the learner write and explain SQL queries, design relational schemas, use PostgreSQL features, reason about query behavior, and practice realistic database tasks.

## Track

- Beginner foundations: tables, rows, columns, keys, data types, `SELECT`, filtering, sorting.
- Core querying: joins, aggregation, grouping, subqueries, CTEs, set operations.
- Data modeling: normalization, constraints, relationships, indexes, schema design.
- PostgreSQL practice: functions/operators, transactions, isolation, MVCC, views, window functions, `EXPLAIN ANALYZE`.
- Professional readiness: query optimization, data integrity, debugging, projects, and interview explanations.

## Evidence

Valid evidence includes reviewed SQL queries, schema designs, query explanations, bug fixes, optimization reasoning, and project work. Generated SQL or lesson exposure is not evidence by itself.

## Practice

Use focused query tasks, explain-output tasks, schema design prompts, Northwind-style scenarios, query debugging, indexing/performance reasoning, and interview-style explanations.
