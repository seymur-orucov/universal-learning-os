## Required Manual Checks Before Use

- Confirm SQL/PostgreSQL generated pack has exactly 25 files.
- Confirm English generated pack has exactly 25 files.
- Confirm launch kits are outside generated pack directories.
- Confirm learner state is under `learners/active-learner/`.
- Run the manual acceptance checklist in `tests/manual-acceptance/ACCEPTANCE_CHECKLIST.md`.

## Start SQL/PostgreSQL Mentor OS

1. Use `exports/generated/project-launch-kits/SQL_POSTGRESQL_PROJECT_SETUP.md`.
2. Paste Project Instructions from `exports/generated/sql-postgresql-standard/PROJECT_INSTRUCTIONS.md`.
3. Upload the SQL/PostgreSQL generated pack files.
4. Start with `exports/generated/project-launch-kits/SQL_POSTGRESQL_STARTUP_PROMPT_READY.md`.

## Start English Mentor OS

1. Use `exports/generated/project-launch-kits/ENGLISH_PROJECT_SETUP.md`.
2. Paste Project Instructions from `exports/generated/english-standard/PROJECT_INSTRUCTIONS.md`.
3. Upload the English generated pack files.
4. Start with `exports/generated/project-launch-kits/ENGLISH_STARTUP_PROMPT_READY.md`.

## Known Risks

- Generated packs are manually maintained and may drift from canonical sources.
- Manual tests are not executable.
- Learner state is Markdown and manually maintained.
- Deleted chats cannot be reconstructed unless summaries or learner state were saved.
- English audio/pronunciation workflows need stronger privacy policy before deeper use.

## Release Status

v0.1.0-rc1 is ready as a manual release candidate for practical project setup and review. It is not a stable release.
