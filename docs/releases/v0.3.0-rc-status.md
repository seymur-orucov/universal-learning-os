# v0.3.0 Release Candidate Status

## Current Release Candidate Status

`v0.3.0` is in release preparation. It is not tagged yet.

The release candidate includes optional learner runtime helpers, automated CLI tests, and the local Studio MVP control panel while preserving generated pack contracts and learner-state boundaries.

## Remaining Before Tag

- Review the v0.3.0 release notes.
- Review the v0.3.0 release checklist.
- Run final validation commands on the intended release commit.
- Confirm no new domain was added.
- Confirm generated pack contracts remain unchanged.
- Confirm Studio remains local-first and CLI-backed.
- Confirm the `v0.3.0` tag has not been created before final approval.

## Release Risks

- Studio is still an MVP and does not have full automated UI coverage.
- Manual acceptance tests remain checklist-based.
- Learner runtime artifacts remain optional scaffolds, not a complete learner-state system.
- Release validation depends on local Node/npm availability.
- Future changes to generated pack sources may require regeneration and validation, but v0.3.0 preparation does not regenerate packs.

## Recommended Final Verification Commands

Run from the repository root:

```sh
node tools/ulos-cli/src/index.js validate
node tools/ulos-cli/src/index.js validate-learner
```
