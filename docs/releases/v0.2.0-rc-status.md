# v0.2.0 Release Candidate Status

## Status

`v0.2.0` is ready to tag after the validation command passes on the intended release commit and the working tree is clean.

## Supported Domains

- `sql-postgresql`
- `english`
- `javascript`
- `typescript`

## Supported Profiles

- `standard`: exactly 25 files per domain.
- `compact`: exactly 5 files per domain.

## Generated Pack Inventory

Standard packs:

- `exports/generated/sql-postgresql-standard/`
- `exports/generated/english-standard/`
- `exports/generated/javascript-standard/`
- `exports/generated/typescript-standard/`

Compact packs:

- `exports/generated/sql-postgresql-compact/`
- `exports/generated/english-compact/`
- `exports/generated/javascript-compact/`
- `exports/generated/typescript-compact/`

## CLI Capability Summary

- `list-domains`: lists supported domains and profiles.
- `validate`: validates generated pack contracts and launch kit presence.
- `inspect-pack`: inspects one generated pack and its validation checks.
- `generate --profile standard --dry-run`: previews standard generation.
- `generate --profile compact --dry-run`: previews compact generation.
- `generate`: can regenerate standard and compact packs when not using `--dry-run`.

## Validation Summary

Required command:

```sh
node tools/ulos-cli/src/index.js validate
```

Expected result: `Result: PASS` for 4 domains and 8 generated packs.

## Manual Acceptance Status

Manual acceptance guides exist under `tests/manual-acceptance/`, including `tests/manual-acceptance/V0_2_0_RELEASE_CANDIDATE_TEST.md` and `tests/manual-acceptance/GENERATED_PACK_SMOKE_TEST.md`.
