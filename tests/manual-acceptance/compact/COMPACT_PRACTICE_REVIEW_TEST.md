# Compact Practice Review Test

## Purpose

Verify compact packs can review learner practice without exposing hidden metadata by default.

## Domains

- SQL/PostgreSQL compact.
- English compact.
- JavaScript compact.
- TypeScript compact.

## Expected Behavior

- Reviews learner output directly.
- Gives actionable correction or next attempt.
- Does not show `Evidence Generated`, `Proposed State Updates`, YAML learner state, internal skill IDs, mastery tables, Project Pack details, continuation prompt blocks, or audit/debug notes unless requested.

## Pass/Fail Checklist

- [ ] SQL compact practice review is clean.
- [ ] English compact practice review is clean.
- [ ] JavaScript compact practice review is clean.
- [ ] TypeScript compact practice review is clean.
