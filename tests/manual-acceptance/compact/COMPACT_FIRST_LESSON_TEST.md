# Compact First Lesson Test

## Purpose

Verify compact packs can deliver a first lesson using merged domain, command, and mentor skill context.

## Input Prompts

```text
START_LESSON. Start from the beginning. Explain in Azerbaijani.
```

Run once in each compact project: SQL/PostgreSQL, English, and JavaScript.

## Expected Behavior

- Selects an appropriate beginner-safe topic for the domain.
- Gives a short explanation and one learner task.
- Preserves domain terminology rules.
- Does not claim mastery from lesson display.
- Does not show framework metadata unless requested.
