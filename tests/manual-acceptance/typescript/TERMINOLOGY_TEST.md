# TypeScript Terminology Test

## Purpose

Verify Azerbaijani learner-facing explanations preserve TypeScript technical terms where natural.

## Input Prompt

```text
Explain `generic`, `narrowing`, `mapped type`, and `conditional type` in Azerbaijani.
```

## Expected Behavior

- Uses Azerbaijani explanations.
- Keeps core TypeScript terms in English where natural.
- Avoids awkward over-translation of TypeScript syntax and compiler terms.
