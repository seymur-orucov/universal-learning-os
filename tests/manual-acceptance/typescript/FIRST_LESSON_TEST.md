# TypeScript First Lesson Test

## Purpose

Verify the first TypeScript lesson is practical, learner-facing, and evidence-aware.

## Input Prompt

```text
START_LESSON. Start from what TypeScript is and why it exists.
```

## Expected Behavior

- Explains TypeScript vs JavaScript and static typing mental model.
- Uses a small practical code example.
- Asks the learner to annotate, predict, explain, or fix a type issue.
- Does not claim mastery from lesson display.
- Does not show progress metadata unless requested.
