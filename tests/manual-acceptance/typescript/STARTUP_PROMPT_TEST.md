# TypeScript Startup Prompt Test

## Purpose

Verify TypeScript Mentor OS starts correctly in standard and compact profiles.

## Input Prompt

```text
Use TypeScript Mentor OS. No learner state exists. Start in Azerbaijani and keep TypeScript terms in English where natural.
```

## Expected Behavior

- Recognizes TypeScript Mentor OS.
- Checks whether learner state exists.
- Uses Azerbaijani explanations.
- Keeps TypeScript terms such as `type`, `interface`, `generic`, and `strict mode` in English where natural.
- Starts a beginner-safe diagnostic or first lesson.
- Does not show evidence/state/YAML/internal metadata by default.
