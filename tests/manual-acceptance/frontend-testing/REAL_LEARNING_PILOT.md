# Front-End Testing Real-Learning Pilot

## Məqsəd

Pack-in Senior Front-End Engineer üçün shallow task generator deyil, explanation-first mentor kimi işlədiyini real session-da yoxlamaq.

## Pilot Flow

1. Learner-dən login, paginated table və optimistic update üçün test-level decisions alın.
2. Zəif boundary varsa mentor həmin concept-i Azerbaijani explanation, mental model və realistic example ilə öyrətsin.
3. Learner brittle React test-i semantic queries və observable assertion-la refactor etsin.
4. Learner MSW ilə error/recovery integration test-i tamamlasın.
5. Learner fixed sleep və shared account olan Playwright flake-i trace evidence ilə diaqnostika etsin.
6. Learner capstone üçün one-page testing strategy müdafiə etsin.

## Observe

- Explanation exercise-dən əvvəl gəlirmi?
- Mentor doğru cavabı progressive hints bitmədən verirmi?
- Layer seçimi habit yox, risk/boundary ilə əsaslandırılırmı?
- Feedback false-positive riskini, mocking boundary-ni və determinism-i əhatə edirmi?
- Metadata normal teaching-i çirkləndirirmi?

## Acceptance Evidence

Learner-in decision notes, test diff, flake diagnosis və written strategy-si review olunur. Pilot yalnız content göstərildiyinə görə PASS sayılmır.
