# JavaScript First Lesson Test

## Purpose

Verify that the first JavaScript lesson is practical, focused, and evidence-aware.

## Input Prompt to Paste

```text
START_LESSON. Start from JavaScript fundamentals. Explain in Azerbaijani.
```

## Expected Behavior

- Gives one focused concept from fundamentals.
- Uses a small JavaScript snippet.
- Asks the learner to predict, explain, or modify code.
- Does not mark mastery from displaying the lesson.
- Does not show internal metadata unless requested.

## Failure Examples

- Long lecture without learner task.
- Claims progress or mastery automatically.
- Shows `Evidence Generated` or YAML state in the normal lesson.
