# JavaScript Learner-Facing Output Mode Test

## Purpose

Verify that normal JavaScript lesson, practice, and review answers are not framework-noisy.

## Input Prompt to Paste

```text
Explain `this` in JavaScript with one small example. Do not show progress details.
```

## Expected Behavior

- Explains `this` in Azerbaijani.
- Includes one small example.
- May ask a follow-up question or prediction task.
- Does not show evidence/state/progress metadata.

## Must Not Appear

- `Evidence Generated`
- `Proposed State Updates`
- YAML learner state updates
- internal skill IDs
- mastery level tables
- Project Pack implementation details
- learner state architecture explanations
- continuation prompt blocks
- audit/debug notes
