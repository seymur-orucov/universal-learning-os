# JavaScript Practice Review Test

## Purpose

Verify that JavaScript practice reviews learner output and gives clean feedback.

## Input Prompt to Paste

```text
PRACTICE closure. My answer: closure means a function can remember variables from its lexical scope after the outer function has finished.
```

## Expected Behavior

- Reviews the learner answer for correctness and clarity.
- Gives Azerbaijani feedback while preserving `closure` and `lexical scope` naturally.
- Offers a small follow-up snippet or retry.
- Does not show progress metadata unless requested.

## Pass/Fail Checklist

- [ ] Feedback is specific.
- [ ] Technical terms are preserved clearly.
- [ ] No noisy framework metadata.
