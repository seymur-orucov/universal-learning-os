# JavaScript Domain Pack

This is the JavaScript pilot domain pack for the Stage 11 domain pack factory model.

It contains reusable subject-specific content for learning JavaScript fundamentals, runtime reasoning, async behavior, browser basics, and frontend interview-style explanation. It is separate from learner state and MUST NOT contain learner-specific progress or mastery records.

## Localization

This domain SHOULD use Azerbaijani learner-facing explanations by default while preserving JavaScript technical terms in English where natural.

## Files

- `DOMAIN_OVERVIEW.md`: domain identity, goals, audience, outcomes, and boundaries.
- `SYLLABUS.md`: staged `javascript.main` learning path.
- `MENTOR_BEHAVIOR.md`: learner-facing teaching behavior and clean output expectations.
- `PRACTICE_MODEL.md`: evidence-producing practice model.
- `ASSESSMENT_MODEL.md`: checkpoint and evidence rules.
- `MASTERY_EXPECTATIONS.md`: domain-specific mastery expectations mapped to framework evidence rules.
- `PROJECT_PACK_SELECTION.md`: generated Project Pack selection and file budget guidance.

## Relationships

- Domain pack rules are defined in `specification/DOMAIN_PACK_SPEC.md`.
- Domain creation process is defined in `docs/domain-pack-factory.md`.
- Project Pack export rules are defined in `exports/project-pack-spec/`.
