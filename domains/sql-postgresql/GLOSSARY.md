# SQL/PostgreSQL Glossary

This glossary follows `specification/LOCALIZATION_SPEC.md`. It preserves important SQL/PostgreSQL terms in English and provides short Azerbaijani explanations for the initial project preference.

## Initial Terms

### SQL

- Terminology language: English
- Short explanation: Relational verilənlər bazaları ilə sorğu yazmaq və məlumatı idarə etmək üçün istifadə olunan dil.

### PostgreSQL

- Terminology language: English
- Short explanation: Açıq mənbəli relational database sistemi.

### table

- Terminology language: English
- Short explanation: Məlumatların rows və columns şəklində saxlandığı struktur.

### row

- Terminology language: English
- Short explanation: Table daxilində bir konkret qeydi ifadə edən məlumat sətri.

### column

- Terminology language: English
- Short explanation: Table daxilində müəyyən məlumat atributunu saxlayan sahə.

### primary key

- Terminology language: English
- Short explanation: Table daxilində hər row üçün unikal identifikator.

### foreign key

- Terminology language: English
- Short explanation: Bir table-dakı dəyəri başqa table-dakı primary key ilə əlaqələndirən constraint.

### SELECT

- Terminology language: English
- Short explanation: Verilənlər bazasından məlumat oxumaq üçün istifadə olunan SQL statement.

### WHERE

- Terminology language: English
- Short explanation: Query nəticələrini şərtə görə filter etmək üçün istifadə olunan clause.

### JOIN

- Terminology language: English
- Short explanation: Əlaqəli table-lardan məlumatı birləşdirmək üçün istifadə olunan əməliyyat.

### GROUP BY

- Terminology language: English
- Short explanation: Rows-u qruplaşdırıb aggregate hesablamalar aparmaq üçün istifadə olunan clause.

### index

- Terminology language: English
- Short explanation: Query axtarışını sürətləndirməyə kömək edə bilən database strukturu.

### transaction

- Terminology language: English
- Short explanation: Bir və ya bir neçə database əməliyyatını vahid, idarə olunan iş vahidi kimi icra edən mexanizm.

### MVCC

- Terminology language: English
- Short explanation: PostgreSQL-də concurrent oxuma və yazma əməliyyatlarını idarə etməyə kömək edən versioning yanaşması.

### EXPLAIN ANALYZE

- Terminology language: English
- Short explanation: PostgreSQL-də query execution plan və real icra məlumatlarını göstərən analiz komandası.
