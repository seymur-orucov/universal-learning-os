# Mastery Model Specifications

The mastery model defines domain-independent expectations for mastery levels and evidence requirements.

## Scope

These documents define framework-level concepts only. Domain packs MAY map them to domain-specific rubrics but MUST NOT weaken evidence requirements.

## Documents

- `core/mastery-model/MASTERY_LEVELS.md`: framework-level mastery scale.
- `core/mastery-model/EVIDENCE_REQUIREMENTS.md`: acceptable evidence before mastery claims.

## Relationships

- Assessment dimensions are defined in `core/learning-engine/ASSESSMENT_MODEL.md`.
- Learner state rules are defined in `specification/STATE_SPEC.md`.
- Lifecycle rules are defined in `specification/LEARNING_LIFECYCLE.md`.
